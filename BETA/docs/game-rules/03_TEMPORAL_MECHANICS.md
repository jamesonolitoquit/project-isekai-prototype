# DSS 03 — Temporal Mechanics & Reincarnation

## 03.1 Paradox Debt Formula (Individual Risk)
Every time a Player influences the "Causal Thread" or "Rewinds" a Snapshot:
`Debt = EventMagnitude * (InformationGained / TemporalDivergence)`

- **Debt 0–25% (Whisper):** Minor d20 roll bias (-1).
- **Debt 26–50% (Bleed):** Environmental NPCs react with "Unrest."
- **Debt 51–75% (Bleach):** "Shadow" entities attract to the Player.
- **Debt 76%+ (Reality Fault):** The Engine triggers a **Vessel Reset**.

## 03.2 Reincarnation (The Ancestral Soul)
When a Vessel is destroyed (Death) or reset (Paradox):
1. **Ancestry Check:** Player selects a new Vessel from **Model A (Canon-Locked)** list.
   - *Constraint:* If the population is extinct, that ancestry is disabled.
2. **Memory Retention:** Skills are reset to **Level 1**, but `AncestralEchoPoints` can be used to "Flash-Learn" (10x XP gain) skills from the previous Vessel.
3. **Heritage Inheritance:**
   - **Faction Capital:** 10% of reputation is inherited by the new Vessel.
   - **Inventory:** Personal items are "Entropic Junk" unless stored in a **Causal Vault**.

## 03.3 Causal Vaults (Legacy Persistence)
- **Vaults:** Static locations in the world immune to `TemporalDivergence`.
- **Function:** Transfers items between Reincarnation cycles.
- **Cost:** High `ResourceStockpile` cost per tick to maintain.
